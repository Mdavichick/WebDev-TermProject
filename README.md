# Build/Run Instructions

Clone Repo
- git clone https://github.com/csc317-sp21/csc317-termproject-Mdavichick.git

CD into application 
- cd .\csc317-termproject-Mdavichick\
- cd .\application\

1. Start server   
- npm start
2. Navigate to:
- localhost:3000/
